About Go Programming Language

Go is a compiled, concurrent, garbage-collected, statically typed language developed at Google. Go is an open source programming language that makes it easy to build simple, reliable, and efficient software

The design of Go

Software engineering guided the design of Go. More than most general-purpose programming languages, Go was designed to address a set of software engineering issues that the Go Authors had been exposed to in the construction of large server software. You may find more about Go Design here: Go at Google: Language Design in the Service of Software Engineering
